# git-check

> A tool for your prompt that provides basic git repo information
> NB: This works only for Bash

## Installation

The manual way:

* Download the script
* Put it in a directory included by `$PATH`
* Run `git-check` to see the docs

## Usage

Add this to `.bashrc`:

```bash
export PS1="$PS1\$(git-check)"
```
